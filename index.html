SQL Injection Lab

직접 해보면서 배우는 SQL 인젝션 실습 환경

빠른 시작

1. 브라우서만으로 실습 (가장 쉽음, 설치 필요 없음)

browser-simulator.html 열기

서버 없이 바로 UNION, 로그인 우회, Blind 시뫄레이션을 체험할 수 있습니다.

2. Python Flask 실습 (강력 추천)

cd python
python db_init.py
python vulnerable_app.py

그 후 http://127.0.0.1:5000 접속

Secure 비교 버전: python secure_app.py → http://127.0.0.1:5001

3. JSP + MySQL 실습 (Tomcat 환경)

먼저 MySQL에서 DB를 생성하세요:

mysql -u root -p < db_setup_mysql.sql

그 후 sqli-lab/jsp/ 안의 파일들을 Tomcat의 webapps/ROOT/sqli/ 로 복사하고 접속:

  • /sqli/vuln-login.jsp Vulnerable
  • /sqli/vuln-search.jsp Vulnerable
  • /sqli/vuln-detail.jsp Vulnerable
  • /sqli/secure-login.jsp Secure

학습 자료

  • README.md — 이론 + 상세 설명 + 방어 방법
  • payloads-cheatsheet.md — 주요 페이로드 모음
  • python/cli_simulator.py — 웹 없이 CLI로 빠르게 실습
